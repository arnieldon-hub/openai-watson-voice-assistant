AI Voice Assistant

A full-stack voice-enabled AI assistant built with Flask, OpenAI, and IBM Watson Speech services. The application supports both text and voice interaction, generating intelligent responses and converting them into audio.

Features
	•	Text-based chatbot powered by OpenAI
	•	Speech-to-text using IBM Watson
	•	Text-to-speech audio responses
	•	Interactive web interface
	•	Real-time response handling

Tech Stack
	•	Python
	•	Flask
	•	OpenAI API
	•	IBM Watson Speech-to-Text (STT)
	•	IBM Watson Text-to-Speech (TTS)
	•	HTML, CSS, JavaScript

How It Works
	1.	User enters text or records voice input
	2.	Audio is converted to text using Watson STT
	3.	Text is sent to OpenAI for processing
	4.	OpenAI generates a response
	5.	Response is converted to speech using Watson TTS
	6.	The result is displayed and played back in the browser
